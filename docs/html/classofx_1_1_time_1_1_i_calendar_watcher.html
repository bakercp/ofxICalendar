ICalendarWatcherEvents ofx::Time::ICalendarWatcher::events

A class containing all events for this watcher. Listener classes can call the utility methods registerAllEvents() or unregisterAllEvents() to register callbacks or manually register for individual events using ofAddListener() and ofRemoveListener().
